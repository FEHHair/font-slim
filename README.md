### 简介
在日常项目中，有一些UI同学经常搞一些其他字体来美化界面，而这种特殊的字体绝大多数情况下仅限于`数字`与`字母`，汉字几乎不会使用，因此，当我们引进整个字体包时，对项目来说是非常冗余的。

font-slim 是一个基于 `fontmin` 封装的字体过滤工具，主要用来过滤无用字体，大幅减小字体包大小。

### 安装依赖

```bash
pnpm install
```
